# MongoDB CRUD Operation using Java
Create Collection,
Populate over Documents,
Update Document based on _id,
Read data in JSON Format,
Delete based on _id






Maven project,
Maven dependency:

<dependencies>
  <dependency>
    <groupId>org.mongodb</groupId>
    <artifactId>mongo-java-driver</artifactId>
    <version>3.11.2</version>
  </dependency>
</dependencies>
